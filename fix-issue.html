SafePal Wallet Issues

Common wallet problems and how to resolve them

Transaction Error

Transaction failed or stuck in pending state. Check network connection and try resending with higher gas fees.

Wallet Sync Failed

Wallet not syncing with blockchain. Try reconnecting or resetting the network connection.

Balance Not Updating

Balance appears incorrect or not updating in real-time. Refresh the wallet or clear cache.

Private Key Access Issue

Unable to access or export private key. Verify you have proper authentication credentials.

Import Wallet Failed

Cannot import wallet from seed phrase or private key. Check format and try again.

Authentication Error

Login failed or authentication token expired. Clear cookies and try logging in again.

Gas Fee Too High

Transaction gas fees are too expensive. Wait for network congestion to reduce or use different network.

Token Not Recognized

Token not showing in wallet or not recognized. Add custom token address manually.

Network Connection Issue

Unable to connect to blockchain network. Check internet connection and network settings.

Account Locked

Account temporarily locked due to multiple failed attempts. Wait 24 hours before retry.

Insufficient Funds

Not enough balance to complete transaction. Add more funds or reduce transaction amount.

Smart Contract Interaction Failed

Failed to interact with smart contract. Verify contract address and transaction parameters.
